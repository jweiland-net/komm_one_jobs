[Komm.ONE](https://www.komm.one/) is a german company located in Baden-WÃ¼rttemberg providing solutions in automated data processing for communal institutions. They also provide multiple secured API endpoints `*.dvvbw.de` for a lot of cities, institutions and topics.

## Introduction

With the TYPO3 extension `komm_one_jobs` you can access just the job database for your city/institution.

## Installation

You can install this extension via Composer with the following command:

```bash
composer req jweiland/komm-one-jobs
```
